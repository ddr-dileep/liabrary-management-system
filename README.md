## Run locally

- Clone the project

```bash
  git clone https://github.com/ddr-dileep/SMS-CHATTING-APP
```

- Go to the project directory

```bash
  cd SMS-CHATTING-APP
```

- Install dependencies

```bash
  yarn install
```

- Run the application

```bash
  yarn run dev
```
